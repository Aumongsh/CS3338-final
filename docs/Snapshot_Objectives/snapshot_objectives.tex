\section*{Start Objective}

The start of the project focuses on establishing the foundation necessary for building the YouTube Comment Comparison Web Application. The main goal of this phase is to clearly define the project's vision and prepare the technical base for development.

\subsection*{Goals}
\begin{itemize}
    \item Define the core purpose and intended functionality of the application.
    \item Conduct brainstorming sessions to refine features and workflow.
    \item Determine team roles and responsibilities.
    \item Establish a version control and collaboration workflow.
\end{itemize}

\subsection*{Technical Setup}
\begin{itemize}
    \item Initialize the React frontend and Node.js/Fastify backend.
    \item Configure dependencies such as Axios and environment variables.
    \item Develop basic frontend pages (homepage and results template).
    \item Create the initial database schema and connect it to backend services.
    \item Set up and test basic API routes to verify frontendâ€“backend communication.
\end{itemize}
