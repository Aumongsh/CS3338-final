\section{How to Use}
\begin{enumerate}
\item Enter the webapp URL into the browser
\item Enter your own Youtube channel URL into 'My Channel'
\item Enter URL of the Youtube channel you want to compare against in 'Compare to: '
\item Specify number of videos you want to compare (1~20)
\item Select whether to compare videos from 'Most Recent' or 'Most Viewed'
\item The webapp will display stats on the total number of shared commenters and per-video w/ the option to sort based on shared commenter count, viewer count, or recency
\end{enumerate}
